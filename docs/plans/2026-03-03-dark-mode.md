# Dark Mode Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a manual dark/light mode toggle to the portfolio site with localStorage persistence and no flash of wrong theme on load.

**Architecture:** `[data-theme="dark"]` on `<html>` activates dark token overrides defined in CSS. A synchronous inline `<script>` at the end of `<head>` reads localStorage and sets the attribute before first paint (preventing FOUC). A toggle button in the header controls the switch.

**Tech Stack:** Vanilla CSS custom properties, vanilla JS, localStorage.

## Task 3: Wire up `initDarkMode()` in client.js

**Files:**
- Modify: `public/client.js`

**Step 1: Add `initDarkMode()` function**

In `public/client.js`, add this function just before the `// --- Initial Load ---` comment (around line 1050):

```javascript
// --- Dark Mode ---

function initDarkMode() {
    const toggleBtn = document.querySelector('.btn-theme-toggle');
    if (!toggleBtn) return;

    function applyTheme(isDark) {
        if (isDark) {
            document.documentElement.setAttribute('data-theme', 'dark');
            toggleBtn.textContent = '☀';
            toggleBtn.setAttribute('aria-label', 'Skift til lyst tema');
        } else {
            document.documentElement.removeAttribute('data-theme');
            toggleBtn.textContent = '☾';
            toggleBtn.setAttribute('aria-label', 'Skift til mørkt tema');
        }
    }

    const savedTheme = localStorage.getItem('theme');
    applyTheme(savedTheme === 'dark');

    toggleBtn.addEventListener('click', () => {
        const isDark = document.documentElement.getAttribute('data-theme') === 'dark';
        const nextIsDark = !isDark;
        applyTheme(nextIsDark);
        if (nextIsDark) {
            localStorage.setItem('theme', 'dark');
        } else {
            localStorage.removeItem('theme');
        }
    });
}
```

**Step 2: Call `initDarkMode()` in the initial load block**

Find the `// --- Initial Load ---` section at the bottom of `client.js`:

```javascript
// --- Initial Load ---
```

Add `initDarkMode();` as the first call:
